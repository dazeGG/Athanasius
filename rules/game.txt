<b>АФАНАСИЙ</b>
Каждому значению карт (типа "2", "7", "Король", "Джокер" (если такие есть)) соответсвует <b>Афанасий</b>. Афанасий начисляется игроку, если ни у кого из других игроков больше нет карт этого значения.

<b>ЦЕЛЬ ИГРЫ</b>
Cобрать больше <b>Афанасиев</b>, чем другие игроки.

<b>ПОДГОТОВКА К ИГРЕ</b>
Игроки выбирают режим игры (определяется вид и количество колод, количество рук). После чего все карты максимально <b>поровну</b> раздаются между всеми игроками.

<b>ХОД ИГРЫ</b>
Игроки ходят поочерёдно. В свой ход игрок пытается угадать карты других игроков. Для этого игрок выбирает другого игрока, и посследовательно пытается угадать наличие карты определённого значения, их количество, соотношение цветов мастей, и масти. Если игрок всё угадал правильно, угаданные карты выходят из игры, то есть больше не принадлежат этому игроку. Игрок, угадавший карты, ходит ещё раз. Ход заканчивается, как только игрок предположил что-то неверно. Все игроки знают, какие вопросы спрашивают другие игроки.

ПРИМЕЧАНИЕ: Игрок может спрашивать только про те карты, которые есть у него на руках.

<b>КОНЕЦ ИГРЫ</b>
Если у игрока больше нет карт на руках, он не может ходить. Игра закнчивается когда у всех игроков заканчиваются карты.

<b>ИГРА В ДВЕ РУКИ</b>
Каждому игроку раздаётся двойной набор карт<i>(по одному на руку)</i>. Наборы между собой не перемешиваются. При попытке угадать учитываются карты только с одной<i>(любой)</i> руки. Считается что игрок предположил <b>верно</b>, если предположение <b>верно</b> хотя бы для одной руки, <b>неверно</b> – если <b>неверно</b> для обеих рук.
<i>(то есть, если у игрока в одной руке одна шестёрка, а в другой три, то на вопрос <b>"у тебя 4  шестёрки?"</b> спрашивающий игрок получит ответ <b>"нет"</b>, а на любой из вопросов <b>"у тебя 1 шестёрка?"</b> и <b>"у тебя 3 шестёрки?"</b> получит ответ <b>"да"</b>).</i>
